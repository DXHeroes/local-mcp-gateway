# Diagrams Directory

## Purpose

Mermaid diagram source files for architecture, flows, and database schema.

## Parent Reference

- **[../AGENTS.md](../AGENTS.md)** - Documentation directory instructions
- **[../../AGENTS.md](../../AGENTS.md)** - Root directory instructions

## Files

- `architecture.mmd` - Application architecture diagram (NestJS backend, React frontend)
- `oauth-flow.mmd` - OAuth 2.1 flow sequence diagram (for MCP servers)
- `mcp-proxy-flow.mmd` - MCP proxy request flow diagram
- `mcp-discovery.mmd` - MCP package auto-discovery flow

## Usage

These Mermaid files are referenced in documentation Markdown files and rendered automatically by GitHub, VS Code, and other Markdown viewers.
