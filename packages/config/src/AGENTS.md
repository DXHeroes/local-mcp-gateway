# Config Source Directory

## Description

Shared build and development configuration presets for the monorepo.

## Contents

- `index.ts` - Main exports
- `typescript.ts` - TypeScript configuration presets
- `tsdown.ts` - tsdown bundler configuration
- `vite.ts` - Vite configuration presets
- `vitest.ts` - Vitest test configuration

## Key Concepts

- **Shared Configs**: Reusable configuration across all packages
- **TypeScript**: Strict mode enabled, ES modules
- **Vite**: Development server and build configuration
- **Vitest**: Test runner configuration with coverage
