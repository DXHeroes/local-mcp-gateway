# UI Source Directory

## Description

Shared UI components built with React, Tailwind CSS, and Radix UI primitives.

## Contents

- `index.ts` - Main exports for all components

## Subdirectories

- **[components/](components/AGENTS.md)** - Reusable UI components (shadcn/ui based)
- **[lib/](lib/AGENTS.md)** - Utility functions (cn for classnames)
- `styles/` - Global styles and Tailwind configuration

## Key Concepts

- **shadcn/ui**: Component library built on Radix UI primitives
- **Tailwind CSS**: Utility-first CSS framework
- **Accessibility**: All components follow WAI-ARIA guidelines
- **Composability**: Components designed for composition
