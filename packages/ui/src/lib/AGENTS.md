# UI Lib Directory

## Description

Utility functions for the UI package.

## Contents

- `utils.ts` - Class name utilities (cn function)

## Key Concepts

- **cn Function**: Combines clsx and tailwind-merge for class name handling
- **Usage**: `cn("base-class", conditional && "conditional-class", props.className)`
- **Tailwind Merge**: Prevents conflicting Tailwind classes
