# UI Components Directory

## Description

Reusable UI components based on shadcn/ui and Radix UI primitives.

## Contents

### Form Components
- `button.tsx` - Button variants (default, destructive, outline, etc.)
- `input.tsx` - Text input field
- `textarea.tsx` - Multi-line text input
- `checkbox.tsx` - Checkbox with label support
- `radio-group.tsx` - Radio button group
- `select.tsx` - Dropdown select with search
- `switch.tsx` - Toggle switch
- `label.tsx` - Form label

### Layout Components
- `card.tsx` - Card container with header/content/footer
- `dialog.tsx` - Modal dialog
- `alert-dialog.tsx` - Confirmation dialog
- `tabs.tsx` - Tabbed interface
- `dropdown-menu.tsx` - Dropdown menu with items

### Feedback Components
- `alert.tsx` - Alert messages
- `badge.tsx` - Status badges
- `toast.tsx` - Toast notifications
- `toaster.tsx` - Toast container
- `use-toast.ts` - Toast hook
- `tooltip.tsx` - Hover tooltips

## Key Concepts

- **Radix UI**: Accessible primitives as foundation
- **Tailwind Variants**: Styling with class variance authority
- **Composition**: Components use slots for customization
- **TypeScript**: Full type safety with generics
