## Key Directories

### Generated (`generated/`)

Auto-generated by Prisma. **DO NOT EDIT MANUALLY.**

- `prisma/` - Prisma Client with TypeScript types
- `zod/` - Zod validation schemas for all models

To regenerate:
```bash
pnpm prisma:generate
```

### Seeds (`seeds/`)

- `seed.ts` - Seed data for default profile and initial setup

To run seeds:
```bash
pnpm prisma:seed
```

### Entry Points

- `database.ts` - Database connection utilities (path resolution, etc.)
- `index.ts` - Package exports (Prisma Client, Zod schemas)

## Child Directories

- **[seeds/AGENTS.md](seeds/AGENTS.md)** - Database seed scripts

## Development Rules

- **Never edit generated files** - Edit `prisma/schema.prisma` instead
- Use Prisma Client for all database operations
- Use Zod schemas for input validation
- Run `pnpm prisma:generate` after schema changes
