# E2E Pages Directory

## Description

Page Object Model (POM) classes for Playwright E2E tests. Each class encapsulates page interactions.

## Contents

- `DebugLogsPage.ts` - Debug logs page interactions
- `McpServersPage.ts` - MCP servers list page interactions
- `ProfileEditPage.ts` - Profile editing page interactions
- `ProfilesPage.ts` - Profiles list page interactions

## Key Concepts

- **Page Object Model**: Encapsulates page selectors and actions
- **Reusability**: Page objects are reused across multiple tests
- **Maintainability**: Selector changes only need updating in one place
- **Playwright**: Uses Playwright locators and assertions
