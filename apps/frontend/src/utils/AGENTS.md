# Utils Directory

## Description

Utility functions for the frontend application.

## Contents

- `promptGenerator.ts` - Generates AI prompts in TOON and Markdown formats

## Key Concepts

- **Prompt Generation**: Converts profile tools to AI-readable formats
- **TOON Format**: Token-optimized format for LLMs
- **Markdown Format**: Human-readable format for documentation
- **Tool Filtering**: Respects enabled/disabled tool settings
