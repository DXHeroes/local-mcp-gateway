1. **Cloud SaaS** - Monthly/annual subscription

## Pricing Philosophy

### Value-Based Pricing

Our pricing reflects:
- **Complexity avoided** - No DIY MCP management
- **Time saved** - Quick setup vs building from scratch
- **Risk reduced** - Proven, maintained solution

### Competitive Positioning

| Comparison | Their Approach | Our Approach |
|------------|---------------|--------------|
| DIY MCP setup | Free (your time) | Starting at $0 |
| API Management | $500+/month | $29-99/month |
| Enterprise tools | $10,000+/year | $2,500+/year |

## Conversion Strategy

```
Free → Pro → Team → Enterprise
  │      │      │
  │      │      └── Team features, shared billing
  │      └── Usage limits, advanced features
  └── Core features, generous limits
```

### Free to Pro Triggers

- Hit profile limit (1 → 10)
- Hit request limit (1K → 50K)
- Need priority support

### Pro to Team Triggers

- Multiple team members
- Audit requirements
- Shared profile management

### Team to Enterprise Triggers

- SSO requirement
- Custom SLA needs
- Compliance requirements
