# Licensing Documentation

License models and terms for Local MCP Gateway.

## Overview

Local MCP Gateway uses a **dual licensing model** that balances open development with sustainable business.

---

## License Types

### Source-Available License

For non-commercial and personal use:
- View and modify source code
- Self-host for personal projects
- Contribute to development
- Learn and experiment

**Restrictions:**
- No commercial use without paid license
- Cannot offer as a hosted service
- Cannot remove license validation

### Commercial License

For business and commercial use:
- Full production deployment rights
- Priority support
- SLA guarantees
- Custom terms available
